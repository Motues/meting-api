# Meting API

基于 [Meting](https://github.com/metowolf/meting) 的 实现的 API

## 快速开始

1. 下载代码
    ```bash
    git clone https://github.com/Motues/meting-api.git
    cd meting-api
    ```
2. 安装依赖，编译代码并启动
   ```
   pnpm install
   pnpm build
   pnpm start
   ```
3. 默认监听 `3000` 端口，可以通过 `.env` 文件进行修改
